# Deploy ke Netlify

[![Deploy to Netlify](https://www.netlify.com/img/deploy/button.svg)](https://app.netlify.com/start/deploy?repository=https://github.com/YOUR_USERNAME/somansapos)

## Quick Deploy Guide

### Method 1: One-Click Deploy (Tercepat)
1. Klik tombol "Deploy to Netlify" di atas
2. Connect dengan GitHub account
3. Konfirmasi repository details
4. Deploy! ✨

### Method 2: Manual Deploy
```bash
# 1. Login to Netlify
npm install -g netlify-cli
netlify login

# 2. Deploy
npm run build
netlify deploy --prod
```

### Method 3: Git Integration
1. Push code ke GitHub/GitLab
2. Import project di [Netlify](https://app.netlify.com/)
3. Auto-deploy on every push
